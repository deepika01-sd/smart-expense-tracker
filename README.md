# Smart Expense Tracker

This is a simple Python project that helps track daily expenses.

I built this project to practice Python programming and understand how real-world applications can store and manage data.

The application has a basic graphical interface made with Tkinter where users can enter their expenses and calculate the total amount spent.

Features:
- Add daily expenses
- Save expense data in a CSV file
- View the total expense
- Simple and easy-to-use interface

Technologies Used:
- Python
- Tkinter
- CSV (for data storage)

How to Run:
1. Download or clone the repository
2. Open the project folder
3. Run the following command in terminal:

python main.py

This project is a beginner-level project created for learning and practice.
